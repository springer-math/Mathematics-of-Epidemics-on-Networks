Epidemics on Networks

This project is intended to provide a range of software that is helpful for users of the forthcoming book Mathematics of network epidemics: from exact to approximate models by Kiss, Miller, and Simon.

At present the project is limited to a (nearly) complete python module which provides tools for simulating epidemics in networks. As the project progresses we anticipate including other languages and hope to also provide tools for numerical solution of the various differential equation approximations.
